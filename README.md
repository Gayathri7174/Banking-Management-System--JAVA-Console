# Banking-Management-System--JAVA-Console

The Java Console-Based Banking Management System is a simple, menu-driven application developed using Core Java. It simulates basic banking operations such as user creation, account management, deposits, withdrawals, and balance inquiries through a console interface.

# System Components
# User Class
Stores user details (e.g., name, user ID)
Represents a bank customer
Each user can have one or more bank accounts

# Account Class
Represents a bank account
Stores account number, account type, and balance
Handles operations such as:
Deposit
Withdrawal
Balance check

# AccountManager Class
Manages users and accounts
Acts as a controller between User and Account classes
Handles:
Creating users
Opening accounts
Performing transactions
Displaying account details

# Features
Create new users
Open bank accounts
Deposit money
Withdraw money
Check account balance
Display user and account details
Menu-driven console interface
